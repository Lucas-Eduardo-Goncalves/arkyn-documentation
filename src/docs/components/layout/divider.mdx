# Divider

import { Divider } from "@arkyn/components";

O componente `Divider` é usado para separar visualmente seções de conteúdo em uma interface. Ele fornece uma linha horizontal ou vertical que ajuda a organizar e estruturar o layout, melhorando a legibilidade e a hierarquia visual.

## Importação

```tsx
import { Divider } from "@arkyn/components";
```

## Propriedades

**`orientation`** - `"horizontal" | "vertical"` (padrão: `"horizontal"`)
Define a orientação do divisor

Todas as outras propriedades HTML padrão de `<div>` também são suportadas.

## Orientações

O componente oferece duas orientações: horizontal e vertical. A orientação horizontal é usada para separar conteúdo verticalmente empilhado, enquanto a vertical separa conteúdo lado a lado.

### Horizontal

A orientação horizontal é a padrão e é usada para separar seções de conteúdo que estão organizadas verticalmente.

<div className="noContent">
  <div>
    <p>Seção 1</p>
    <Divider />
    <p>Seção 2</p>
    <Divider orientation="horizontal" />
    <p>Seção 3</p>
  </div>
</div>

```tsx
<div>
  <p>Seção 1</p>
  <Divider />
  <p>Seção 2</p>
  <Divider orientation="horizontal" />
  <p>Seção 3</p>
</div>
```

### Vertical

A orientação vertical é usada para separar elementos que estão dispostos horizontalmente, como em barras de navegação ou grupos de botões.

<div className="noContent">
  <div style={{ display: "flex", alignItems: "center", gap: "16px" }}>
    <span>Item 1</span>
    <Divider orientation="vertical" />
    <span>Item 2</span>
    <Divider orientation="vertical" />
    <span>Item 3</span>
  </div>
</div>

```tsx
<div style={{ display: "flex", alignItems: "center", gap: "16px" }}>
  <span>Item 1</span>
  <Divider orientation="vertical" />
  <span>Item 2</span>
  <Divider orientation="vertical" />
  <span>Item 3</span>
</div>
```

## Casos de Uso Comuns

O `Divider` é extremamente útil em diversos cenários de interface para organizar e estruturar o conteúdo.

    <p>Endereço: São Paulo, SP</p>

    <Divider />

    <h3>Preferências</h3>
    <p>Tema: Claro</p>
    <p>Idioma: Português</p>

  </div>
</div>

```tsx
<div>
```

### Grupos de Botões

Use divisores verticais para separar visualmente grupos de botões ou ações relacionadas.

<div className="noContent">
  <div style={{ display: "flex", alignItems: "center", gap: "8px" }}>
    <button
      style={{
        padding: "8px 16px",
        border: "1px solid #d1d5db",
        borderRadius: "6px",
        background: "white",
      }}
    >
      Salvar
    </button>
    <button
      style={{
        padding: "8px 16px",
        border: "1px solid #d1d5db",
        borderRadius: "6px",
        background: "white",
      }}

      Cancelar
    </button>

    <Divider orientation="vertical" />

    <button
      style={{
        padding: "8px 16px",
        border: "1px solid #dc2626",
        borderRadius: "6px",
        background: "white",
        color: "#dc2626",
      }}
    >
      Excluir
    </button>

  </div>
</div>

```tsx
<div style={{ display: "flex", alignItems: "center", gap: "8px" }}>
  <button>Salvar</button>
  <button>Cancelar</button>

  <Divider orientation="vertical" />

  <button>Excluir</button>
</div>
```

## Acessibilidade

O `Divider` é implementado como um elemento `<div>` e é puramente decorativo por padrão. Para melhor acessibilidade:

- É automaticamente ignorado por leitores de tela por ser decorativo
- Não interfere na navegação por teclado
- Mantém a estrutura semântica do conteúdo

## Notas Importantes

- O divisor horizontal funciona melhor em layouts com `display: block` ou `flex-direction: column`
- O divisor vertical requer um container com `display: flex` para funcionar corretamente
- O componente herda todas as propriedades padrão de um elemento HTML `<div>`
- É recomendado usar o divisor com moderação para evitar poluição visual
- O componente é puramente visual e não afeta a estrutura semântica do HTML
