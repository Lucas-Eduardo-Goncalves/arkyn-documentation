O componente `IconButton` é uma versão compacta do botão que renderiza apenas um ícone. É ideal para interfaces onde o espaço é limitado ou quando o ícone por si só é suficientemente claro para transmitir a ação desejada.

## Importação

```tsx
import { IconButton } from "@arkyn/components";
```

## Tamanhos

Os tamanhos disponíveis são `lg`, `md`, `sm` e `xs`. Cada tamanho oferece uma altura e padding diferentes, com os ícones sendo redimensionados automaticamente (xs: 12px, sm: 16px, md: 20px, lg: 24px).

  <IconButton icon={Plus} aria-label="Adicionar" size="sm" />
  <IconButton icon={Plus} aria-label="Adicionar" size="xs" />
</div>

```tsx

```

## Variantes

As variantes disponíveis são `solid`, `outline`, `ghost` e `invisible`. Cada variante oferece um estilo visual diferente, permitindo criar hierarquias e enfatizar diferentes tipos de ações na interface.

<div className="noContent row">
  <IconButton icon={Settings} aria-label="Configurações" variant="solid" />
  <IconButton icon={Settings} aria-label="Configurações" variant="outline" />
  <IconButton icon={Settings} aria-label="Configurações" variant="ghost" />
  <IconButton icon={Settings} aria-label="Configurações" variant="invisible" />
</div>

```tsx
<IconButton icon={Settings} aria-label="Configurações" variant="solid" />
<IconButton icon={Settings} aria-label="Configurações" variant="outline" />
<IconButton icon={Settings} aria-label="Configurações" variant="ghost" />
<IconButton icon={Settings} aria-label="Configurações" variant="invisible" />
```

## Esquemas de Cores

Os esquemas de cores disponíveis são `primary`, `success`, `warning`, `danger` e `info`. Cada esquema transmite um significado semântico específico e ajuda os usuários a entender o tipo de ação que será executada.

</div>

## Botão Desabilitado

O estado de não ativação do botão indica que ele não pode ser clicado ou interagido. É ativado pela propriedade `disabled` ou quando estiver em estado de carregamento `isLoading`.

<div className="noContent row">
  <IconButton icon={Plus} aria-label="Adicionar" disabled />
  <IconButton icon={Heart} aria-label="Curtir" disabled scheme="success" />
  <IconButton icon={Star} aria-label="Favoritar" disabled variant="outline" />
</div>

```tsx
<IconButton icon={Plus} aria-label="Adicionar" disabled />
<IconButton icon={Heart} aria-label="Curtir" disabled scheme="success" />
<IconButton icon={Star} aria-label="Favoritar" disabled variant="outline" />
```

## Estado de Carregamento

O estado de carregamento é usado para indicar que uma operação está em andamento, mostrando um spinner animado no lugar do ícone original e desabilitando automaticamente o botão.

<div className="noContent row">
  <IconButton icon={Save} aria-label="Salvando" isLoading />
  <IconButton icon={Upload} aria-label="Enviando" isLoading scheme="success" />
  <IconButton icon={Trash2} aria-label="Excluindo" isLoading scheme="danger" />
</div>

```tsx
<IconButton icon={Save} aria-label="Salvando" isLoading />
<IconButton icon={Upload} aria-label="Enviando" isLoading scheme="success" />
<IconButton icon={Trash2} aria-label="Excluindo" isLoading scheme="danger" />
```

## Propriedades

**`icon`** - `LucideIcon` (obrigatório)
Ícone a ser renderizado dentro do botão

**`aria-label`** - `string` (obrigatório)
Rótulo acessível para leitores de tela

**`isLoading`** - `boolean` (padrão: `false`)
Controla o estado de carregamento com spinner

**`size`** - `"xs" | "sm" | "md" | "lg"` (padrão: `"md"`)
Tamanho do botão

**`variant`** - `"solid" | "outline" | "ghost" | "invisible"` (padrão: `"solid"`)
Variante visual do botão

**`scheme`** - `"primary" | "success" | "warning" | "danger" | "info"` (padrão: `"primary"`)
Esquema de cores do botão

## Acessibilidade

O `IconButton` foi projetado com acessibilidade em mente:

- A propriedade `aria-label` é obrigatória para fornecer contexto aos usuários de leitores de tela
- O botão é automaticamente focável via teclado
- Estados visuais claros para hover, focus e disabled
- Suporte completo a navegação por teclado

```tsx
// ✅ Correto - sempre forneça aria-label
<IconButton icon={Save} aria-label="Salvar documento" />

// ❌ Incorreto - aria-label é obrigatório
<IconButton icon={Save} />
```

## Notas Importantes

- A propriedade `aria-label` é obrigatória para acessibilidade
- Quando `isLoading` é `true`, o botão é automaticamente desabilitado
- Os ícones são redimensionados automaticamente com base no tamanho do botão (xs: 12px, sm: 16px, md: 20px, lg: 24px)
- O componente não aceita `children` - apenas o ícone especificado é renderizado
- O componente herda todas as propriedades padrão de um elemento HTML `<button>`
- Durante o carregamento, um spinner da biblioteca Lucide React substitui o ícone original
- Todos os manipuladores de eventos padrão do HTML são suportados (onClick, onFocus, onBlur, etc.)
