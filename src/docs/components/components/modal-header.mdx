# ModalHeader

import {
  ModalContainer,
  ModalFooter,
  ModalHeader,
  Button,
} from "@arkyn/components";
import { ModalExample } from "../../_examples/modal.example";

ModalHeader renders the top section of a modal with optional built-in close button.

```tsx
import { ModalHeader } from "@arkyn/components";
```

## Basic Usage

<div className="noContent">
  <ModalExample>
    <ModalHeader>Modal Title</ModalHeader>
    <main style={{ padding: 30 }}>
      <p>Are you sure you want to continue?</p>
    </main>
  </ModalExample>
</div>

```tsx
<ModalContainer>
    <ModalHeader>Modal Title</ModalHeader>
    <main style={{ padding: 30 }}>
        <p>Are you sure you want to continue?</p>
    </main>
</ModalContainer>
```

## Close Button Visibility

Set `showCloseButton` to control whether the header renders the close action.

<div className="noContent column">
  <ModalExample>
    <ModalHeader>With Close Button</ModalHeader>
    <main style={{ padding: 30 }}>
      <p>Are you sure you want to continue?</p>
    </main>
  </ModalExample>
  <ModalExample>
    <ModalHeader showCloseButton={false}>Without Close Button</ModalHeader>
    <main style={{ padding: 30 }}>
      <p>Are you sure you want to continue?</p>
    </main>
  </ModalExample>
</div>

```tsx
<ModalContainer>
    <ModalHeader>
        With Close Button
    </ModalHeader>
    <main style={{ padding: 30 }}>
        <p>Are you sure you want to continue?</p>
    </main>
</ModalContainer>
<ModalContainer>
    <ModalHeader showCloseButton={false}>
        Without Close Button
    </ModalHeader>
    <main style={{ padding: 30 }}>
        <p>Are you sure you want to continue?</p>
    </main>
</ModalContainer>
```

```tsx
<ModalContainer>

```

## Inside ModalContainer

Use ModalHeader inside ModalContainer so the close button can call modal context.

<div className="noContent">
  <ModalExample >
    <div>
      <ModalHeader>
        Confirm Action
      </ModalHeader>

      <main style={{ padding: 30 }}>
        <p>Review this action before continuing.</p>
      </main>

      <ModalFooter>
        <Button >Cancel</Button>
        <Button >Continue</Button>
      </ModalFooter>
    </div>

  </ModalExample>
</div>

```tsx
<ModalContainer isVisible={isOpen} makeInvisible={() => setIsOpen(false)}>
    <div>
        <ModalHeader>Confirm Action</ModalHeader>

        <main style={{ padding: 30 }}>
            <p>Review this action before continuing.</p>
        </main>

        <ModalFooter>
            <Button onClick={() => setIsOpen(false)}>Cancel</Button>
            <Button onClick={handleConfirm}>Continue</Button>
        </ModalFooter>
    </div>
</ModalContainer>
```

## Properties

**`showCloseButton`** - `boolean` (default: `true`)
Whether to render the close button in the header.

**`className`** - `string`
Additional CSS classes merged with the base header class.

**`children`** - `ReactNode`
Header content, usually title and optional subtitle.

**`...rest`** - `HTMLAttributes<HTMLElement>`
Any other valid HTML attributes for the root `header` element.
