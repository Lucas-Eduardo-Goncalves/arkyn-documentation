# TableBody

import {
  TableBody,
  TableContainer,
  TableHeader,
} from "@arkyn/components";

TableBody renders table rows and provides an automatic empty state when no rows exist.

```tsx
import { TableBody } from "@arkyn/components";
```

## Basic Usage

<div className="noContent column">
  <TableContainer>
    <TableHeader>
      <th>Name</th>
      <th>Email</th>
    </TableHeader>

    <TableBody>
      <tr>
        <td>John Doe</td>
        <td>john@example.com</td>
      </tr>
      <tr>
        <td>Jane Smith</td>
        <td>jane@example.com</td>
      </tr>
    </TableBody>
  </TableContainer>
</div>

```tsx
<TableBody>
  <tr>
    <td>John Doe</td>
    <td>john@example.com</td>
  </tr>
  <tr>
    <td>Jane Smith</td>
    <td>jane@example.com</td>
  </tr>
</TableBody>
```

## Empty State

When no children are provided, TableBody renders a full-width empty message row.

<div className="noContent column">
  <TableContainer>
    <TableHeader>
      <th>User</th>
      <th>Status</th>
    </TableHeader>

    <TableBody emptyMessage="No users found. Please add one to continue." />
  </TableContainer>
</div>

```tsx
<TableBody emptyMessage="No users found. Please add one to continue." />
```

## Dynamic Rows

Use mapped data rows while keeping the same fallback behavior.

<div className="noContent column">
  <TableContainer>
    <TableHeader>
      <th>Product</th>
      <th>Price</th>
      <th>Stock</th>
    </TableHeader>

    <TableBody className="striped-rows">
      <tr>
        <td>Keyboard</td>
        <td>$99.00</td>
        <td>12</td>
      </tr>
      <tr>
        <td>Monitor</td>
        <td>$299.00</td>
        <td>7</td>
      </tr>
    </TableBody>
  </TableContainer>
</div>

```tsx
<TableBody className="striped-rows" emptyMessage="No products available">
  {products.map((product) => (
    <tr key={product.id}>
      <td>{product.name}</td>
      <td>{product.price}</td>
      <td>{product.stock}</td>
    </tr>
  ))}
</TableBody>
```

## Properties

**`emptyMessage`** - `string` (default: `"Nenhum dado adicionado."`)
Message displayed when there are no row children.

**`children`** - `ReactNode`
Table rows rendered inside the body section.

**`className`** - `string`
Additional CSS classes merged with the base body class.

**`...rest`** - `HTMLAttributes<HTMLTableSectionElement>`
Any other valid HTML attributes for the root `tbody` element.
