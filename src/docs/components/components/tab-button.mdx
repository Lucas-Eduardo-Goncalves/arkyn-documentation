# TabButton

import {
  TabContainer,
  TabButton,
  AlertContainer,
  AlertContent,
  AlertDescription,
  AlertIcon,
  AlertTitle,
} from "@arkyn/components";

`TabButton` is the interactive tab trigger used inside `TabContainer`, handling active state and selection changes.

```tsx
import { TabContainer, TabButton } from "@arkyn/components";
```

## Basic Usage

<div className="noContent column">
  <TabContainer>
    <TabButton value="home">Home</TabButton>
    <TabButton value="about">About</TabButton>
    <TabButton value="contact">Contact</TabButton>
  </TabContainer>
</div>

```tsx
<TabContainer>
    <TabButton value="home">Home</TabButton>
    <TabButton value="about">About</TabButton>
    <TabButton value="contact">Contact</TabButton>
</TabContainer>
```

## Rich Content

You can pass custom content as `children` for complex labels.

<div className="noContent column">
  <TabContainer defaultValue="dashboard">
    <TabButton value="dashboard">
      <div>
        <strong>Dashboard</strong>
      </div>
    </TabButton>
    <TabButton value="analytics">
      <div>
        <strong>Analytics</strong>
      </div>
    </TabButton>
  </TabContainer>
</div>

```tsx
<TabContainer defaultValue="dashboard">
    <TabButton value="dashboard">
        <div>
            <strong>Dashboard</strong>
        </div>
    </TabButton>
    <TabButton value="analytics">
        <div>
            <strong>Analytics</strong>
        </div>
    </TabButton>
</TabContainer>
```

## Click Handling

`TabButton` supports custom click handlers while still updating tab context.

<div className="noContent column">
  <TabContainer>
    <TabButton value="profile">Profile</TabButton>
    <TabButton value="settings">Settings</TabButton>
  </TabContainer>
</div>

```tsx
<TabContainer onChange={(value) => console.log("Tab changed:", value)}>
    <TabButton value="profile" onClick={() => console.log("Profile tab clicked")}>
        Profile
    </TabButton>
    <TabButton value="settings">Settings</TabButton>
</TabContainer>
```

## Disabled

Disable individual tabs with `disabled`, or disable all tabs through the parent container.

<div className="noContent column">
  <TabContainer>
    <TabButton value="available">Available</TabButton>
    <TabButton value="coming-soon" disabled>
      Coming Soon
    </TabButton>
  </TabContainer>

  <TabContainer disabled>
    <TabButton value="locked-a">Locked A</TabButton>
    <TabButton value="locked-b">Locked B</TabButton>
  </TabContainer>
</div>

```tsx
<TabContainer>
    <TabButton value="available">Available</TabButton>
    <TabButton value="coming-soon" disabled>
        Coming Soon
    </TabButton>
</TabContainer>

<TabContainer disabled>
    <TabButton value="locked-a">Locked A</TabButton>
    <TabButton value="locked-b">Locked B</TabButton>
</TabContainer>
```

## Properties

**`children`** - `ReactNode`
Content displayed inside the tab button.

**`value`** - `string`
Unique identifier used to activate and track the current tab.

**`disabled`** - `boolean`
Disables this specific tab button.

**`className`** - `string`
Additional classes for the button element.

**`onClick`** - `(event: MouseEvent<HTMLButtonElement>) => void`
Custom click handler called alongside internal tab selection.

**`aria-label`** - `string`
Optional accessibility label.

**`title`** - `string`
Native title attribute.

**`id`** - `string`
Optional id for the button element.

**`...rest`** - `Omit<ButtonHTMLAttributes<HTMLButtonElement>, "children" | "value" | "type">`
Any other valid button attributes are forwarded except excluded ones.
