# GoogleAnalytics

import { GoogleAnalytics } from "@arkyn/components/googleAnalytics";
import { AlertContainer } from "@arkyn/components/alertContainer";
import { AlertContent } from "@arkyn/components/alertContent";
import { AlertDescription } from "@arkyn/components/alertDescription";
import { AlertIcon } from "@arkyn/components/alertIcon";
import { AlertTitle } from "@arkyn/components/alertTitle";

`GoogleAnalytics` injects Google Analytics 4 scripts into the document head and initializes `gtag` with the provided measurement id.

```tsx
import { GoogleAnalytics } from "@arkyn/components/googleAnalytics";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

## Basic Usage

<div className="noContent">
  <GoogleAnalytics measurementId="G-XXXXXXXXXX" />
</div>

```tsx
<GoogleAnalytics measurementId="G-XXXXXXXXXX" />
```

## Environment Behavior

By default, the component renders only in production. Use `showInDevMode` when you need local validation.

<div className="noContent column">
  <GoogleAnalytics measurementId="G-XXXXXXXXXX" />
  <GoogleAnalytics measurementId="G-XXXXXXXXXX" showInDevMode />
</div>

```tsx
// Production only (default)
<GoogleAnalytics measurementId="G-XXXXXXXXXX" />

// Force rendering in development
<GoogleAnalytics measurementId="G-XXXXXXXXXX" showInDevMode />
```

## Client-Side Script Injection

The component delegates script creation to internal utilities that build the async `gtag.js` URL and inline bootstrap script.

<div className="noContent">
  <GoogleAnalytics measurementId="G-TEST123456" showInDevMode />
</div>

```tsx
<GoogleAnalytics measurementId="G-TEST123456" showInDevMode />
```

## Properties

**`measurementId`** - `string`
Google Analytics 4 Measurement ID used to configure `gtag`.

**`showInDevMode`** - `boolean` (default: `false`)
If true, renders and initializes analytics outside production.
