# Alert

import { AlertContainer } from "@arkyn/components/alertContainer";
import { AlertContent } from "@arkyn/components/alertContent";
import { AlertDescription } from "@arkyn/components/alertDescription";
import { AlertIcon } from "@arkyn/components/alertIcon";
import { AlertTitle } from "@arkyn/components/alertTitle";

The Alert family of components displays contextual feedback messages. It is composed of `AlertContainer` (root wrapper and schema styling), `AlertContent` (groups title and description), `AlertIcon` (schema-aware icon), `AlertTitle` (heading text), and `AlertDescription` (descriptive text).

```tsx
import { AlertContainer } from "@arkyn/components/alertContainer";
import { AlertContent } from "@arkyn/components/alertContent";
import { AlertDescription } from "@arkyn/components/alertDescription";
import { AlertIcon } from "@arkyn/components/alertIcon";
import { AlertTitle } from "@arkyn/components/alertTitle";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

## Basic Usage

<div className="noContent column">
  <AlertContainer schema="info">
    <AlertIcon />
    <AlertContent>
      <AlertDescription>System information message.</AlertDescription>
    </AlertContent>
  </AlertContainer>
</div>

```tsx
<AlertContainer schema="info">
    <AlertIcon />
    <AlertContent>
        <AlertDescription>System information message.</AlertDescription>
    </AlertContent>
</AlertContainer>
```

## Schemas

Use `schema` on `AlertContainer` to define semantic meaning and visual style. `AlertIcon` automatically renders the matching icon for the schema.

<div className="noContent column">
  <AlertContainer schema="success">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Success</AlertTitle>
      <AlertDescription>Your operation completed successfully.</AlertDescription>
    </AlertContent>
  </AlertContainer>

<AlertContainer schema="danger">
  <AlertIcon />
  <AlertContent>
    <AlertTitle>Error</AlertTitle>
    <AlertDescription>
      Something failed while processing your request.
    </AlertDescription>
  </AlertContent>
</AlertContainer>

<AlertContainer schema="warning">
  <AlertIcon />
  <AlertContent>
    <AlertTitle>Warning</AlertTitle>
    <AlertDescription>
      Please review this action before continuing.
    </AlertDescription>
  </AlertContent>
</AlertContainer>

  <AlertContainer schema="info">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Information</AlertTitle>
      <AlertDescription>Updates are available for this feature.</AlertDescription>
    </AlertContent>
  </AlertContainer>
</div>

```tsx
<AlertContainer schema="success">...</AlertContainer>
<AlertContainer schema="danger">...</AlertContainer>
<AlertContainer schema="warning">...</AlertContainer>
<AlertContainer schema="info">...</AlertContainer>
```

## Title-aware Layout

When `AlertTitle` exists in the alert tree, content is left-aligned. Without `AlertTitle`, the layout becomes centered automatically.

<div className="noContent column">
  <AlertContainer schema="success">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Success</AlertTitle>
      <AlertDescription>You are now a premium user.</AlertDescription>
    </AlertContent>
  </AlertContainer>

  <AlertContainer schema="info">
    <AlertIcon />
    <AlertContent>
      <AlertDescription>Short single-line informational message.</AlertDescription>
    </AlertContent>
  </AlertContainer>
</div>

```tsx
<AlertContainer schema="success">
    <AlertIcon />
    <AlertContent>
        <AlertTitle>Success</AlertTitle>
        <AlertDescription>You are now a premium user.</AlertDescription>
    </AlertContent>
</AlertContainer>

<AlertContainer schema="info">
    <AlertIcon />
    <AlertContent>
        <AlertDescription>Short single-line informational message.</AlertDescription>
    </AlertContent>
</AlertContainer>
```

## Schema-based Icons

`AlertIcon` changes automatically according to the container schema: `CheckCircle2` for `success`, `XCircle` for `danger`, `AlertTriangle` for `warning`, and `Info` for `info`.

<div className="noContent column">
  <AlertContainer schema="success">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Success</AlertTitle>
      <AlertDescription>CheckCircle2 icon is rendered.</AlertDescription>
    </AlertContent>
  </AlertContainer>
  <AlertContainer schema="danger">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Error</AlertTitle>
      <AlertDescription>XCircle icon is rendered.</AlertDescription>
    </AlertContent>
  </AlertContainer>
  <AlertContainer schema="warning">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Warning</AlertTitle>
      <AlertDescription>AlertTriangle icon is rendered.</AlertDescription>
    </AlertContent>
  </AlertContainer>
  <AlertContainer schema="info">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Info</AlertTitle>
      <AlertDescription>Info icon is rendered.</AlertDescription>
    </AlertContent>
  </AlertContainer>
</div>

```tsx
<AlertContainer schema="success"><AlertIcon /></AlertContainer>
<AlertContainer schema="danger"><AlertIcon /></AlertContainer>
<AlertContainer schema="warning"><AlertIcon /></AlertContainer>
<AlertContainer schema="info"><AlertIcon /></AlertContainer>
```

## Properties

### AlertContainer

**`schema`** - `"success" | "danger" | "warning" | "info"`
Required visual and semantic style of the alert.

**`children`** - `React.ReactNode`
Alert content, typically composed with `AlertIcon`, `AlertContent`, `AlertTitle`, and `AlertDescription`.

**Standard HTML div attributes** - `HTMLAttributes<HTMLDivElement>`
All native `div` props are supported, including `className`, `id`, and accessibility attributes.

### AlertContent

**`children`** - `React.ReactNode`
Content rendered inside the alert content area.

**Standard HTML div attributes** - `HTMLAttributes<HTMLDivElement>`
All native `div` props are supported, including `className`, `id`, and accessibility attributes.

### AlertTitle

**`children`** - `React.ReactNode`
Title content rendered in the alert.

**Standard HTML div attributes** - `HTMLAttributes<HTMLDivElement>`
All native `div` props are supported, including `className`, `id`, and accessibility attributes.

### AlertDescription

**`children`** - `React.ReactNode`
Description content rendered inside the alert.

**Standard HTML div attributes** - `HTMLAttributes<HTMLDivElement>`
All native `div` props are supported, including `className`, `id`, and accessibility attributes.

### AlertIcon

**Icon props** - `LucideProps`
Supports all Lucide icon properties such as `size`, `strokeWidth`, `color`, and `className`.
