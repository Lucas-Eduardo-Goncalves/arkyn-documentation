# MultiSelect

import { MultiSelect } from "@arkyn/components/multiSelect";
import { AlertContainer } from "@arkyn/components/alertContainer";
import { AlertContent } from "@arkyn/components/alertContent";
import { AlertDescription } from "@arkyn/components/alertDescription";
import { AlertIcon } from "@arkyn/components/alertIcon";
import { AlertTitle } from "@arkyn/components/alertTitle";

import { Globe } from "lucide-react";

`MultiSelect` allows selecting multiple options from a dropdown with optional search, inline selected chips, and form-ready serialization.

```tsx
import { MultiSelect } from "@arkyn/components/multiSelect";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

## Basic Usage

<div className="noContent">
  <MultiSelect
    name="categories"
    options={[
      { label: "Technology", value: "tech" },
      { label: "Design", value: "design" },
      { label: "Marketing", value: "marketing" },
    ]}
  />
</div>

```tsx
<MultiSelect
    name="categories"
    options={[
        { label: "Technology", value: "tech" },
        { label: "Design", value: "design" },
        { label: "Marketing", value: "marketing" },
    ]}
/>
```

## Controlled And Close-On-Select Behavior

Use controlled values with `onChange`, and set `closeOnSelect` to close the menu after each selection.

<div className="noContent column">
  <MultiSelect
    name="tags"
    label="Tags"
    value={["frontend", "docs"]}
    closeOnSelect={false}
    options={[
      { label: "Frontend", value: "frontend" },
      { label: "Backend", value: "backend" },
      { label: "Docs", value: "docs" },
    ]}
  />

  <MultiSelect
    name="departments"
    label="Departments"
    prefix="Dept:"
    closeOnSelect
    defaultValue={["hr", "it"]}
    variant="underline"
    options={[
      { label: "HR", value: "hr" },
      { label: "IT", value: "it" },
      { label: "Operations", value: "ops" },
    ]}
  />
</div>

```tsx
<MultiSelect
    name="tags"
    label="Tags"
    value={selectedTags}
    onChange={setSelectedTags}
    onSearch={handleSearch}
    closeOnSelect={false}
    options={tagOptions}
/>

<MultiSelect
    name="departments"
    label="Departments"
    prefix="Dept:"
    closeOnSelect
    defaultValue={["hr", "it"]}
    variant="underline"
    options={departmentOptions}
/>
```

## Properties

**`name`** - `string`
Field name used in form submission.

**`options`** - `{ label: string; value: string }[]`
Selectable options list.

**`id`** - `string`
Custom unique id for the component.

**`value`** - `string[]`
Controlled array of selected option values.

**`defaultValue`** - `string[]` (default: `[]`)
Initial selected values for uncontrolled usage.

**`showAsterisk`** - `boolean`
Displays required indicator on the label.

**`label`** - `string`
Optional label shown above the field.

**`errorMessage`** - `string`
Custom error message displayed below the component.

**`placeholder`** - `string` (default: `"Selecione..."`)
Text shown when no option is selected.

**`notFoundText`** - `string` (default: `"Sem opções disponíveis"`)
Text shown when no options match the search.

**`className`** - `string`
Additional class names for the outer wrapper.

**`disabled`** - `boolean` (default: `false`)
Disables interaction.

**`readOnly`** - `boolean` (default: `false`)
Prevents changes while preserving display.

**`isLoading`** - `boolean` (default: `false`)
Shows loading state and disables interaction.

**`isSearchable`** - `boolean` (default: `false`)
Enables search field within options panel.

**`closeOnSelect`** - `boolean` (default: `false`)
Closes dropdown after selecting an option.

**`onSearch`** - `(value: string) => void`
Called when search input changes.

**`onChange`** - `(value: string[]) => void`
Called when selected values change.

**`onFocus`** - `() => void`
Called when the component gains focus.

**`onBlur`** - `(e: FocusEvent<HTMLDivElement>) => void`
Called when the component loses focus.

**`size`** - `"md" | "lg"` (default: `"md"`)
Controls component size.

**`variant`** - `"solid" | "outline" | "underline"` (default: `"solid"`)
Defines visual style.

**`prefix`** - `string | LucideIcon`
Text or icon shown before selected marks.

**`leftIcon`** - `LucideIcon`
Optional icon shown on the left side.

**`optionMaxHeight`** - `number`
Maximum height for options dropdown.
