# TableCaption

import {
  TableBody,
  TableCaption,
  TableContainer,
  TableHeader,
} from "@arkyn/components";

TableCaption displays a title or descriptive heading for table content.

```tsx
import { TableCaption } from "@arkyn/components";
```

## Basic Usage

<div className="noContent column">
  <TableContainer>
    <TableCaption>User Management</TableCaption>

    <TableHeader>
      <th>Name</th>
      <th>Email</th>
    </TableHeader>

    <TableBody>
      <tr>
        <td>John Doe</td>
        <td>john@example.com</td>
      </tr>
    </TableBody>
  </TableContainer>
</div>

```tsx
<TableContainer>
  <TableCaption>User Management</TableCaption>

  <TableHeader>
    <th>Name</th>
    <th>Email</th>
  </TableHeader>

  <TableBody>
    <tr>
      <td>John Doe</td>
      <td>john@example.com</td>
    </tr>
  </TableBody>
</TableContainer>
```

## Rich Caption Content

Caption can render structured content such as heading and metadata.

<div className="noContent column">
  <TableContainer>
    <TableCaption>
      <h3>Employee Directory</h3>
      <p>Updated today</p>
    </TableCaption>

    <TableHeader>
      <th>ID</th>
      <th>Name</th>
      <th>Department</th>
    </TableHeader>

    <TableBody>
      <tr>
        <td>EMP-01</td>
        <td>Maria</td>
        <td>Design</td>
      </tr>
    </TableBody>
  </TableContainer>
</div>

```tsx
<TableContainer>
  <TableCaption>
    <h3>Employee Directory</h3>
    <p>Updated: {new Date().toLocaleDateString()}</p>
  </TableCaption>

  <TableHeader>
    <th>ID</th>
    <th>Name</th>
    <th>Department</th>
  </TableHeader>

  <TableBody>
    <tr>
      <td>EMP-01</td>
      <td>Maria</td>
      <td>Design</td>
    </tr>
  </TableBody>
</TableContainer>
```

## Custom Class Name

Use custom classes to emphasize caption style.

<div className="noContent column">
  <TableContainer>
    <TableCaption className="highlighted-caption">
      Sales Report - Q4 2024
    </TableCaption>

    <TableHeader>
      <th>Product</th>
      <th>Revenue</th>
    </TableHeader>

    <TableBody>
      <tr>
        <td>Plan A</td>
        <td>$12,000</td>
      </tr>
    </TableBody>
  </TableContainer>
</div>

```tsx
<TableCaption className="highlighted-caption">
  Sales Report - Q4 2024
</TableCaption>
```

## Properties

**`children`** - `ReactNode`
Caption content rendered inside the caption container.

**`className`** - `string`
Additional CSS classes merged with the base caption class.

**`...rest`** - `HTMLAttributes<HTMLElement>`
Any other valid HTML attributes for the root `caption` element.
