# AlertTitle

import {
  AlertContainer,
  AlertContent,
  AlertDescription,
  AlertIcon,
  AlertTitle,
} from "@arkyn/components";

The `AlertTitle` component renders the heading text for an alert message.

```tsx
import {
  AlertContainer,
  AlertContent,
  AlertDescription,
  AlertIcon,
  AlertTitle,
} from "@arkyn/components";
```

## Basic Usage

<div className="noContent column">
  <AlertContainer schema="success">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Success</AlertTitle>
      <AlertDescription>Your profile has been updated.</AlertDescription>
    </AlertContent>
  </AlertContainer>
</div>

```tsx
<AlertContainer schema="success">
  <AlertIcon />
  <AlertContent>
    <AlertTitle>Success</AlertTitle>
    <AlertDescription>Your profile has been updated.</AlertDescription>
  </AlertContent>
</AlertContainer>
```

## Different Schemas

Use the title with any alert schema to highlight the message context.

<div className="noContent column">
  <AlertContainer schema="danger">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Request failed</AlertTitle>
      <AlertDescription>Try again in a few minutes.</AlertDescription>
    </AlertContent>
  </AlertContainer>
  <AlertContainer schema="info">
    <AlertIcon />
    <AlertContent>
      <AlertTitle>Update available</AlertTitle>
      <AlertDescription>A new release can be installed now.</AlertDescription>
    </AlertContent>
  </AlertContainer>
</div>

```tsx
<AlertContainer schema="danger">...</AlertContainer>
<AlertContainer schema="info">...</AlertContainer>
```

## Properties

**`children`** - `React.ReactNode`
Title content rendered in the alert.

**Standard HTML div attributes** - `HTMLAttributes<HTMLDivElement>`
All native `div` props are supported, including `className`, `id`, and accessibility attributes.
