# Drawer

import { DrawerContainer } from "@arkyn/components/drawerContainer";
import { DrawerHeader } from "@arkyn/components/drawerHeader";
import { AlertContainer } from "@arkyn/components/alertContainer";
import { AlertContent } from "@arkyn/components/alertContent";
import { AlertDescription } from "@arkyn/components/alertDescription";
import { AlertIcon } from "@arkyn/components/alertIcon";
import { AlertTitle } from "@arkyn/components/alertTitle";

import { DrawerExample } from "../../_examples/drawer.example";

The Drawer family renders an animated side panel. `DrawerContainer` is the animated panel that opens over the page content with overlay click-to-close behavior, and `DrawerHeader` renders its top section with an optional built-in close action.

```tsx
import { DrawerContainer } from "@arkyn/components/drawerContainer";
import { DrawerHeader } from "@arkyn/components/drawerHeader";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

## Basic Usage

<div className="noContent">
  <DrawerExample title="Drawer Example">
    <p>Drawer content</p>
  </DrawerExample>
</div>

```tsx
<DrawerContainer isVisible={isOpen} makeInvisible={() => setIsOpen(false)}>
    <p>Drawer content</p>
</DrawerContainer>
```

## Orientation

Use `orientation` on `DrawerContainer` to control the side from which the drawer enters.

<div className="noContent column" style={{ width: "100%", gap: 16 }}>
  <DrawerExample title="Left Drawer" orientation="left">
    <p>Left drawer</p>
  </DrawerExample>

  <DrawerExample title="Right Drawer" orientation="right">
    <p>Right drawer</p>
  </DrawerExample>
</div>

```tsx
<DrawerContainer
    isVisible={isOpen}
    makeInvisible={() => setIsOpen(false)}
    orientation="left"
>
    <p>Left drawer</p>
</DrawerContainer>

<DrawerContainer
    isVisible={isOpen}
    makeInvisible={() => setIsOpen(false)}
    orientation="right"
>
    <p>Right drawer</p>
</DrawerContainer>
```

## Composing With DrawerHeader

Combine `DrawerContainer` with `DrawerHeader` to create a reusable panel layout with close action.

<div className="noContent">
  <DrawerExample title="Menu" orientation="right">
    <div style={{ padding: 16 }}>
      <p>Home</p>
      <p>About</p>
      <p>Contact</p>
    </div>
  </DrawerExample>
</div>

```tsx
<DrawerContainer
    isVisible={isDrawerOpen}
    makeInvisible={() => setIsDrawerOpen(false)}
    orientation="right"
>
    <DrawerHeader>
        <h2>Menu</h2>
    </DrawerHeader>

    <div className="drawer-content">
        <ul>
            <li>Home</li>
            <li>About</li>
            <li>Contact</li>
        </ul>
    </div>
</DrawerContainer>
```

## Close Button

By default, `DrawerHeader` renders a close button that calls the drawer context action. Use `showCloseButton={false}` to hide it.

<div className="noContent column" style={{ width: "100%", gap: 16 }}>
  <DrawerExample title="Navigation" />
  <DrawerExample title="Settings" />
</div>

```tsx
<DrawerHeader>Navigation</DrawerHeader>
<DrawerHeader showCloseButton={false}>Settings</DrawerHeader>
```

## Custom Header Content

You can pass any header content to `DrawerHeader`, including logos, title groups, and helper text.

<div className="noContent">
  <DrawerExample
    className="custom-header"
    headerChildren={
      <div>
        <h3 style={{ margin: 0 }}>My App</h3>
        <p style={{ margin: 0 }}>Welcome to your workspace</p>
      </div>
    }
  >
    <p>Drawer content</p>
  </DrawerExample>
</div>

```tsx
<DrawerContainer isVisible={isOpen} makeInvisible={() => setIsOpen(false)}>
    <DrawerHeader className="custom-header">
        <div className="header-content">
            <h3 style={{ margin: 0 }}>My App</h3>
            <p style={{ margin: 0 }}>Welcome to your workspace</p>
        </div>
    </DrawerHeader>
</DrawerContainer>
```

## Properties

### DrawerContainer

**`isVisible`** - `boolean`
Controls whether the drawer is rendered and animated in view.

**`makeInvisible`** - `() => void`
Callback used to close the drawer, including overlay click behavior.

**`orientation`** - `"left" | "right"` (default: `"left"`)
Defines the side from which the drawer slides in.

**`children`** - `ReactNode`
Drawer content rendered inside the animated content panel.

**`className`** - `string`
Adds custom classes to the root aside wrapper.

**`id`** - `string`
Sets the id attribute on the root aside wrapper.

**`style`** - `CSSProperties`
Applies inline styles to the root aside wrapper.

**`role`** - `AriaRole`
Defines an ARIA role for the root aside wrapper when needed.

**`...rest`** - `HTMLAttributes<HTMLElement>`
Any other valid native attributes for the root element are forwarded.

### DrawerHeader

**`showCloseButton`** - `boolean` (default: `true`)
Controls visibility of the built-in close button.

**`children`** - `ReactNode`
Content displayed in the header area.

**`className`** - `string`
Adds custom classes to the root header element.

**`id`** - `string`
Sets the id attribute on the root header element.

**`style`** - `CSSProperties`
Applies inline styles to the root header element.

**`role`** - `AriaRole`
Defines an ARIA role for accessibility scenarios.

**`...rest`** - `HTMLAttributes<HTMLElement>`
Any other valid native attributes for a header element are forwarded.
