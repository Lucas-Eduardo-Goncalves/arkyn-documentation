# TableHeader

import {
  TableBody,
  TableContainer,
  TableHeader,
} from "@arkyn/components";

TableHeader renders table column headers with built-in spacing row.

```tsx
import { TableHeader } from "@arkyn/components";
```

## Basic Usage

<div className="noContent column">
  <TableContainer>
    <TableHeader>
      <th>Name</th>
      <th>Email</th>
      <th>Actions</th>
    </TableHeader>

    <TableBody>
      <tr>
        <td>John Doe</td>
        <td>john@example.com</td>
        <td>Edit</td>
      </tr>
    </TableBody>
  </TableContainer>
</div>

```tsx
<TableHeader>
    <th>Name</th>
    <th>Email</th>
    <th>Actions</th>
</TableHeader>
```

## Header with Custom Columns

You can render any header structure needed for your dataset.

<div className="noContent column">
  <TableContainer>
    <TableHeader>
      <th>Product</th>
      <th>Price</th>
      <th>Stock</th>
      <th>Status</th>
    </TableHeader>

    <TableBody>
      <tr>
        <td>Mouse</td>
        <td>$49.00</td>
        <td>34</td>
        <td>Available</td>
      </tr>
    </TableBody>
  </TableContainer>
</div>

```tsx
<TableHeader>
    <th>Product</th>
    <th>Price</th>
    <th>Stock</th>
    <th>Status</th>
</TableHeader>
```

## Custom Class Name

Apply custom classes to control typography and header colors.

<div className="noContent column">
  <TableContainer>
    <TableHeader className="custom-header">
      <th>ID</th>
      <th>User</th>
      <th>Status</th>
    </TableHeader>

    <TableBody>
      <tr>
        <td>10</td>
        <td>Ana</td>
        <td>Active</td>
      </tr>
    </TableBody>
  </TableContainer>
</div>

```tsx
<TableHeader className="custom-header">
    <th>ID</th>
    <th>User</th>
    <th>Status</th>
</TableHeader>
```

## Properties

**`children`** - `ReactNode`
Header cells, typically one or more `th` elements.

**`className`** - `string`
Additional CSS classes merged with the base header class.

**`...rest`** - `HTMLAttributes<HTMLTableSectionElement>`
Any other valid HTML attributes for the root `thead` element.
