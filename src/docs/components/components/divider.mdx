# Divider

import { Divider } from "@arkyn/components/divider";
import { AlertContainer } from "@arkyn/components/alertContainer";
import { AlertContent } from "@arkyn/components/alertContent";
import { AlertDescription } from "@arkyn/components/alertDescription";
import { AlertIcon } from "@arkyn/components/alertIcon";
import { AlertTitle } from "@arkyn/components/alertTitle";

`Divider` is a lightweight separator used to visually split related content blocks in horizontal or vertical layouts.

```tsx
import { Divider } from "@arkyn/components/divider";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

## Basic Usage

<div className="noContent">
  <Divider />
</div>

```tsx
<Divider />
```

## Orientation

Use `orientation` to switch between horizontal section breaks and vertical separators.

<div className="noContent column" style={{ width: "100%", gap: 16 }}>
  <div style={{ width: "100%" }}>
    <p>Horizontal separator</p>
    <Divider orientation="horizontal" />
  </div>

  <div
    style={{
      display: "flex",
      alignItems: "center",
      justifyContent: "center",
      gap: 12,
      height: 48,
    }}
  >
    <span>Left</span>
    <Divider orientation="vertical" style={{ height: "100%" }} />
    <span>Right</span>
  </div>
</div>

```tsx
<Divider orientation="horizontal" />

<div style={{ display: "flex", alignItems: "center", gap: 12, height: 48 }}>
    <span>Left</span>
    <Divider orientation="vertical" style={{ height: "100%" }} />
    <span>Right</span>
</div>
```

## Custom Styling

Pass `className` and native `div` attributes to adapt spacing, color, and layout behavior.

<div className="noContent column" style={{ width: "100%", gap: 16 }}>
  <Divider className="my-custom-divider" />

  <div style={{ width: "100%" }}>
    <h4 style={{ marginBottom: 8 }}>Section 1</h4>
    <p style={{ marginBottom: 8 }}>Content for section 1</p>
    <Divider />
    <h4 style={{ marginTop: 8, marginBottom: 8 }}>Section 2</h4>
    <p>Content for section 2</p>
  </div>
</div>

```tsx
<Divider className="my-custom-divider" />

<div>
    <h2>Section 1</h2>
    <p>Content for section 1</p>
    <Divider />
    <h2>Section 2</h2>
    <p>Content for section 2</p>
</div>
```

## Properties

**`orientation`** - `"horizontal" | "vertical"` (default: `"horizontal"`)
Defines whether the divider is rendered as a horizontal or vertical separator.

**`className`** - `string`
Adds custom classes to the root element.

**`id`** - `string`
Sets the id attribute on the root element.

**`style`** - `CSSProperties`
Applies inline styles to the root element.

**`role`** - `AriaRole`
Defines the ARIA role for accessibility scenarios.

**`...rest`** - `HTMLAttributes<HTMLDivElement>`
Any other valid native `div` attributes are forwarded to the root element.
