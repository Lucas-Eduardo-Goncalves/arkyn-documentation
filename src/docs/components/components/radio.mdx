# Radio

import { RadioGroup } from "@arkyn/components/radioGroup";
import { RadioBox } from "@arkyn/components/radioBox";
import { AlertContainer } from "@arkyn/components/alertContainer";
import { AlertContent } from "@arkyn/components/alertContent";
import { AlertDescription } from "@arkyn/components/alertDescription";
import { AlertIcon } from "@arkyn/components/alertIcon";
import { AlertTitle } from "@arkyn/components/alertTitle";

The Radio family renders single-choice option sets. `RadioGroup` manages shared selection state for a set of options and integrates with form validation, while `RadioBox` is an individual option item that must be rendered inside `RadioGroup` to receive selection context.

```tsx
import { RadioGroup } from "@arkyn/components/radioGroup";
import { RadioBox } from "@arkyn/components/radioBox";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

## Basic Usage

<div className="noContent column">
  <RadioGroup name="gender">
    <RadioBox value="male">Male</RadioBox>
    <RadioBox value="female">Female</RadioBox>
    <RadioBox value="other">Other</RadioBox>
  </RadioGroup>
</div>

```tsx
<RadioGroup name="gender">
    <RadioBox value="male">Male</RadioBox>
    <RadioBox value="female">Female</RadioBox>
    <RadioBox value="other">Other</RadioBox>
</RadioGroup>
```

## Controlled And Default Values

Use `defaultValue` for uncontrolled flows or `value` with `onChange` for controlled state.

<div className="noContent column">
  <RadioGroup name="language" label="Language" defaultValue="en">
    <RadioBox value="en">English</RadioBox>
    <RadioBox value="es">Spanish</RadioBox>
    <RadioBox value="pt">Portuguese</RadioBox>
  </RadioGroup>

  <RadioGroup name="theme" label="Theme" value="dark">
    <RadioBox value="light">Light</RadioBox>
    <RadioBox value="dark">Dark</RadioBox>
    <RadioBox value="auto">Auto</RadioBox>
  </RadioGroup>
</div>

```tsx
<RadioGroup name="language" label="Language" defaultValue="en">
    <RadioBox value="en">English</RadioBox>
    <RadioBox value="es">Spanish</RadioBox>
    <RadioBox value="pt">Portuguese</RadioBox>
</RadioGroup>

<RadioGroup
    name="theme"
    label="Theme"
    value={selectedTheme}
    onChange={(value) => setSelectedTheme(value)}
>
    <RadioBox value="light">Light</RadioBox>
    <RadioBox value="dark">Dark</RadioBox>
    <RadioBox value="auto">Auto</RadioBox>
</RadioGroup>
```

## Error State

Provide `errorMessage` directly on `RadioGroup` or rely on form context errors. Error styling is inherited by every child `RadioBox`.

<div className="noContent column">
  <RadioGroup
    name="subscription"
    label="Choose your plan"
    showAsterisk
    errorMessage="Please select a subscription plan"
  >
    <RadioBox value="basic">Basic Plan</RadioBox>
    <RadioBox value="premium">Premium Plan</RadioBox>
  </RadioGroup>
</div>

```tsx
<RadioGroup
    name="subscription"
    label="Choose your plan"
    showAsterisk
    errorMessage="Please select a subscription plan"
>
    <RadioBox value="basic">Basic Plan</RadioBox>
    <RadioBox value="premium">Premium Plan</RadioBox>
</RadioGroup>
```

## Properties

### RadioGroup

**`name`** - `string`
Field name used for hidden form submission.

**`label`** - `string`
Optional label shown above the group.

**`showAsterisk`** - `boolean`
Displays required indicator in the label.

**`errorMessage`** - `string`
Custom error message shown below the group.

**`value`** - `string`
Controlled selected value.

**`defaultValue`** - `string` (default: `""`)
Initial selected value for uncontrolled usage.

**`onChange`** - `(value: string) => void`
Called when selection changes.

**`size`** - `"sm" | "md" | "lg"` (default: `"md"`)
Default size applied to child radio boxes.

**`disabled`** - `boolean`
Disables selection for all options in the group.

**`className`** - `string`
Additional class names for the wrapper.

**`...rest`** - `Omit<HTMLAttributes<HTMLDivElement>, "onChange">`
Any other valid native `div` attributes are forwarded except native `onChange`.

### RadioBox

**`value`** - `string`
Unique option value used for selection matching.

**`isError`** - `boolean`
Optional error style override for the option.

**`size`** - `"sm" | "md" | "lg"`
Size override for this option.

**`children`** - `ReactNode`
Content rendered beside the radio indicator.

**`disabled`** - `boolean`
Disables this option.

**`className`** - `string`
Additional classes applied to the option wrapper.

**`onClick`** - `(event: MouseEvent<HTMLButtonElement>) => void`
Click handler called before internal selection update.

**`onFocus`** - `(event: FocusEvent<HTMLButtonElement>) => void`
Focus handler called before internal selection update.

**`...rest`** - `ButtonHTMLAttributes<HTMLButtonElement>`
Any other valid native button attributes are forwarded.
