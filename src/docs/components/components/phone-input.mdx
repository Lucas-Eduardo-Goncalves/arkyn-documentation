# PhoneInput

import { PhoneInput } from "@arkyn/components/phoneInput";
import { AlertContainer } from "@arkyn/components/alertContainer";
import { AlertContent } from "@arkyn/components/alertContent";
import { AlertDescription } from "@arkyn/components/alertDescription";
import { AlertIcon } from "@arkyn/components/alertIcon";
import { AlertTitle } from "@arkyn/components/alertTitle";

`PhoneInput` is an international phone field with country selector, dynamic mask handling, and hidden normalized value submission.

```tsx
import { PhoneInput } from "@arkyn/components/phoneInput";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

## Basic Usage

<div className="noContent">
  <PhoneInput name="phone" label="Phone" />
</div>

```tsx
<PhoneInput name="phone" label="Phone" />
```

<div className="noContent column">
  <PhoneInput name="phoneMd" label="Phone (md)" size="md" />
  <PhoneInput name="phoneLg" label="Phone (lg)" size="lg" />
</div>

```tsx
<PhoneInput name="phoneMd" label="Phone (md)" size="md" />
<PhoneInput name="phoneLg" label="Phone (lg)" size="lg" />
```

<PhoneInput name="solidPhone" label="Solid" variant="solid" />

## Country Selection And Search

<div className="noContent">
  <PhoneInput
    name="countryPhone"
    label="Country phone"
    searchCountryPlaceholder="Search country"
    notFoundCountryText="No countries found"
  />
</div>

```tsx
<PhoneInput
    name="countryPhone"
    label="Country phone"
    searchCountryPlaceholder="Search country"

/>
```

## Default Values And Controlled Usage

Use `defaultValue` for uncontrolled initialization or pass `value` with `onChange` for controlled behavior.

<div className="noContent column">
  <PhoneInput
    name="defaultPhone"
    label="Default phone"
    defaultValue="+55 11 99999-9999"
    defaultCountryIso="BR"
  />

  <PhoneInput
    name="controlledPhone"
    label="Controlled phone"
    value="(11) 98888-7777"
  />
</div>

```tsx
<PhoneInput
    name="defaultPhone"
    label="Default phone"
    defaultValue="+55 11 99999-9999"
    defaultCountryIso="BR"
/>

<PhoneInput
    name="controlledPhone"
    label="Controlled phone"
    value={phoneValue}
    onChange={(normalizedPhone) => setPhoneValue(normalizedPhone)}
/>
```

## Properties

**`name`** - `string`
Field name used for hidden form submission.

**`id`** - `string`
Custom unique id used for label/input association.

**`disabled`** - `boolean` (default: `false`)
Disables country selection and phone editing.

**`readOnly`** - `boolean` (default: `false`)
Keeps display active while preventing changes.

**`errorMessage`** - `string`
Custom error message shown below the field.

**`label`** - `string`
Optional label displayed above the field.

**`showAsterisk`** - `boolean`
Displays required indicator in the label.

**`isLoading`** - `boolean` (default: `false`)
Blocks interactions while loading.

**`size`** - `"md" | "lg"` (default: `"md"`)
Controls component dimensions.

**`variant`** - `"solid" | "outline"` (default: `"solid"`)
Defines visual style of the container.

**`className`** - `string`
Additional class names for the outer wrapper.

**`defaultValue`** - `string`
Initial phone value used for uncontrolled behavior.

**`notFoundCountryText`** - `string` (default: `"Nenhum país encontrado"`)
Text shown when no country matches search.

**`searchCountryPlaceholder`** - `string` (default: `"Pesquisar país"`)
Placeholder text for country search input.

**`defaultCountryIso`** - `(typeof countries)[number]["iso"]`
Default country ISO used for initial country selection.

**`onChange`** - `(e: string) => void`
Called with normalized phone value when input changes.

**`value`** - `string`
Controlled phone value.
