# ModalContainer

import {
  AlertContainer,
  AlertContent,
  AlertDescription,
  AlertIcon,
  AlertTitle,
  ModalContainer,
  ModalFooter,
  ModalHeader,
  Button,
} from "@arkyn/components";

import { ModalExample } from "../../_examples/modal.example";

ModalContainer renders an animated overlay modal with content area and outside-click close behavior.

```tsx
import { ModalContainer } from "@arkyn/components";
```

## Basic Usage

<div className="noContent">
  <ModalExample>
    <div style={{ minWidth: 280, padding: 30 }}>Modal content</div>
  </ModalExample>
</div>

```tsx
const [isOpen, setIsOpen] = useState(false);

<ModalContainer isVisible={isOpen} makeInvisible={() => setIsOpen(false)}>
    <p style={{ padding: 30 }}>Modal content</p>
</ModalContainer>;
```

## Header and Footer Composition

Use `ModalHeader` and `ModalFooter` to build consistent modal structure.

<div className="noContent">
  <ModalExample>
    <ModalHeader>
      Confirm Action
    </ModalHeader>

    <main style={{ padding: 30 }}>
      <p>Are you sure you want to continue?</p>
    </main>

    <ModalFooter>
      <Button type="button">Cancel</Button>
      <Button type="button">Confirm</Button>
    </ModalFooter>

  </ModalExample>
</div>

<ModalContainer isVisible={isOpen} makeInvisible={() => setIsOpen(false)}>

## Visibility Control

Modal visibility is controlled by the `isVisible` prop and closed via `makeInvisible`.

<div className="noContent">
  <ModalExample>
    <div style={{ padding: 30 }}>Click the overlay to close</div>
  </ModalExample>
</div>

```tsx
const [isOpen, setIsOpen] = useState(false);

<button type="button" onClick={() => setIsOpen(true)}>
    Open modal
</button>

<ModalContainer isVisible={isOpen} makeInvisible={() => setIsOpen(false)}>
    <div  style={{ padding: 30 }}>Modal body</div>
</ModalContainer>
```

## Properties

**`isVisible`** - `boolean`
Controls whether the modal is rendered and visible.

**`makeInvisible`** - `() => void`
Callback used to close the modal.

**`className`** - `string` (default: `""`)
Additional CSS classes merged with the base modal container class.

**`children`** - `ReactNode`
Content rendered inside the modal content area.

**`...rest`** - `HTMLAttributes<HTMLElement>`
Any other valid HTML attributes for the root `aside` element.
