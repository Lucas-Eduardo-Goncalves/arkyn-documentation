The Modal family renders an animated overlay dialog. `ModalContainer` is the animated overlay with content area and outside-click close behavior, `ModalHeader` renders its top section with an optional built-in close button, and `ModalFooter` renders the action area with configurable content alignment.

```tsx
import { ModalContainer } from "@arkyn/components/modalContainer";
import { ModalHeader } from "@arkyn/components/modalHeader";
import { ModalFooter } from "@arkyn/components/modalFooter";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

## Basic Usage

<div className="noContent">
  <ModalExample>
    <div style={{ minWidth: 280, padding: 30 }}>Modal content</div>
  </ModalExample>
</div>

```tsx
const [isOpen, setIsOpen] = useState(false);

<ModalContainer isVisible={isOpen} makeInvisible={() => setIsOpen(false)}>
    <p style={{ padding: 30 }}>Modal content</p>
</ModalContainer>;
```

## Header and Footer Composition

Use `ModalHeader` and `ModalFooter` together with `ModalContainer` to build consistent modal structure.

## Visibility Control

Modal visibility is controlled by the `isVisible` prop and closed via `makeInvisible`.

<div className="noContent">
  <ModalExample>
    <div style={{ padding: 30 }}>Click the overlay to close</div>
  </ModalExample>
</div>

```tsx
const [isOpen, setIsOpen] = useState(false);

<button type="button" onClick={() => setIsOpen(true)}>
    Open modal
</button>

<ModalContainer isVisible={isOpen} makeInvisible={() => setIsOpen(false)}>
    <div  style={{ padding: 30 }}>Modal body</div>
</ModalContainer>
```

## Properties

### ModalContainer

**`isVisible`** - `boolean`
Controls whether the modal is rendered and visible.

**`makeInvisible`** - `() => void`
Callback used to close the modal.

**`className`** - `string` (default: `""`)
Additional CSS classes merged with the base modal container class.

**`children`** - `ReactNode`
Content rendered inside the modal content area.

**`...rest`** - `HTMLAttributes<HTMLElement>`
Any other valid HTML attributes for the root `aside` element.

### ModalHeader

**`showCloseButton`** - `boolean` (default: `true`)
Whether to render the close button in the header.

**`className`** - `string`
Additional CSS classes merged with the base header class.

**`children`** - `ReactNode`
Header content, usually title and optional subtitle.

**`...rest`** - `HTMLAttributes<HTMLElement>`
Any other valid HTML attributes for the root `header` element.

### ModalFooter

**`alignment`** - `"left" | "center" | "right" | "between" | "around"` (default: `"right"`)
Defines how footer content is aligned.

**`className`** - `string`
Additional CSS classes merged with the base footer class.

**`children`** - `ReactNode`
Footer action elements such as Buttons.

**`...rest`** - `HTMLAttributes<HTMLElement>`
Any other valid HTML attributes for the root `footer` element.
