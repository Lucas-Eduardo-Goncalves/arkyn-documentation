# MaskedInput

import { MaskedInput } from "@arkyn/components/maskedInput";
import { AlertContainer } from "@arkyn/components/alertContainer";

`MaskedInput` is a text field with declarative input masks, supporting labels, validation feedback, decorators, and loading behavior.

```tsx
import { MaskedInput } from "@arkyn/components/maskedInput";
```

Learn how subpath and root imports differ in [How do I use imports](/docs/guides/how-do-i-use-imports).

    replacement={{ _: /\d/ }}
    placeholder="(00) 00000-0000"
  />

## Properties

**`name`** - `string`
Field name used in form submissions.

**`mask`** - `string`
Mask pattern string that defines the input format.

**`replacement`** - `string | Replacement`
Replacement rule used to map editable mask positions.

**`separate`** - `boolean`
Controls separation behavior for mask literals.

**`showMask`** - `boolean`
Controls whether mask placeholders are shown.

**`label`** - `string`
Optional label shown above the field.

**`errorMessage`** - `string`
Custom error text rendered below the field.

**`isLoading`** - `boolean` (default: `false`)
Shows spinner feedback and disables interaction.

**`size`** - `"md" | "lg"` (default: `"md"`)
Controls field sizing and icon proportions.

**`variant`** - `"solid" | "outline" | "underline"` (default: `"solid"`)
Defines the visual container style.

**`prefix`** - `string | LucideIcon`
Text or icon rendered before masked content.

**`suffix`** - `string | LucideIcon`
Text or icon rendered after masked content.

**`showAsterisk`** - `boolean`
Displays required indicator on the label.

**`leftIcon`** - `LucideIcon`
Icon rendered on the left side of the input.

**`rightIcon`** - `LucideIcon`
Icon rendered on the right side of the input.

**`value`** - `string`
Controlled masked value.

**`defaultValue`** - `string`
Initial value for uncontrolled usage.

**`disabled`** - `boolean`
Disables user interaction.

**`readOnly`** - `boolean`
Prevents editing while keeping focus behavior.

**`onFocus`** - `(event: FocusEvent<HTMLInputElement>) => void`
Called when the input receives focus.

**`onBlur`** - `(event: FocusEvent<HTMLInputElement>) => void`
Called when the input loses focus.

**`id`** - `string`
Custom id used to connect label and input.

**`placeholder`** - `string`
Placeholder text displayed when there is no value.

**`title`** - `string`
Native title attribute on the clickable section wrapper.

**`style`** - `CSSProperties`
Inline styles applied to the clickable section wrapper.

**`className`** - `string`
Additional class names for the outer field wrapper.

**`...rest`** - `InputHTMLAttributes<HTMLInputElement>`
Other valid native input attributes are forwarded except `size`, `prefix`, `name`, and `type` typing overrides from this component.
