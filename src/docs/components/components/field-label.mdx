# FieldLabel

import {
  AlertContainer,
  AlertContent,
  AlertDescription,
  AlertIcon,
  AlertTitle,
  FieldLabel,
  Textarea,
} from "@arkyn/components";

FieldLabel renders a form label with optional required-field indicator.

```tsx
import { FieldLabel } from "@arkyn/components";
```

## Basic Usage

<div className="noContent">
  <FieldLabel htmlFor="username">Username</FieldLabel>
</div>

```tsx
<FieldLabel htmlFor="username">Username</FieldLabel>
```

## Required Indicator

Enable `showAsterisk` to display a required-field marker.

<div className="noContent">
  <FieldLabel htmlFor="email" showAsterisk>
    Email Address
  </FieldLabel>
</div>

```tsx
<FieldLabel htmlFor="email" showAsterisk>
    Email Address
</FieldLabel>
```

## Custom Class Name

Use custom classes for visual adjustments when needed.

<div className="noContent">
  <FieldLabel htmlFor="password" className="custom-label" showAsterisk>
    Password
  </FieldLabel>
</div>

```tsx
<FieldLabel htmlFor="password" className="custom-label" showAsterisk>
    Password
</FieldLabel>
```

## Properties

**`showAsterisk`** - `boolean` (default: `false`)
Whether to display an asterisk (`*`) to indicate a required field.

**`className`** - `string` (default: `""`)
Additional CSS classes merged into the base class.

**`...rest`** - `LabelHTMLAttributes<HTMLLabelElement>`
Any other valid HTML attributes for the root `label` element.
